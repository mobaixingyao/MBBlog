## 一个开源的现代风格个人博客模板

### 前言

本项目为纯前端静态网页，无后端，可部署到Cloudflare等网页托管网站的喵

你可以查看下文快速配置来了解如何配置成你的专属博客

最后声明一下，本人只是一个初中生，趁着寒假利用AI做出来的一个简易HTML项目，本人之前也没有怎么接触过html和js

如果你有什么意见也欢迎在issues中指出喵

### 快速配置:

打开index.html文件，在第4-9行配置网页图标和标题

在第81-82行配置名称和个性签名

在第85-96行配置设置跳转个人账号的链接

在第105-205行配置文章

在214配置页脚

在639配置播放器音频链接
