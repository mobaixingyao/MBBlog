# MBBlog
## 一个开源的现代风格个人博客模板

本项目为纯前端静态网页，无后端，可部署到Cloudflare等网页托管网站的喵

如果要写文章可以直接到index.html里面编辑

在index.html里面可以使用markdown语法写文章，使用了JS打开时会自动渲染成HTML

还有，如果要写文章的话要鸡德设置每一篇文章的id。

其他的我都在网页文件里的两篇文章里说了，可以自行去看

最后声明一下，本人只是一个初中生，趁着寒假利用AI做出来的一个简易HTML项目，本人之前也没有怎么接触过html和js

如果有做的不好的地方还请多多担待
